# Implementación de Pruebas Automatizadas con Selenium WebDriver

Este proyecto corresponde a una simulación de pruebas automatizadas
realizada para el curso de Calidad del Software.

## Tecnologías utilizadas
- Java
- Selenium WebDriver
- JUnit
- NetBeans

## Casos de prueba automatizados
Se implementaron los siguientes casos de prueba:
1. Verificación de ingreso de texto en la barra de búsqueda
2. Verificación de ingreso de texto alternativo
3. Verificación de ingreso de términos técnicos
4. Verificación de ejecución de búsqueda
5. Verificación de interacción continua
6. Validación final de ingreso de texto
